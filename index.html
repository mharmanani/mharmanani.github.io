PERSONAL PROJECTS
Unbeatable Tic-Tac-Toe

A game of tic tac toe against an unbeatable artifical agent.

The AI makes use of the minimax function to recursively enumerate all possible outcomes, and determines which scenario hurts the player the most and benefits it the most.

I explored a little slice of classical artifical intelligence with this project, tried my hand at game searching, heuristic functions, and recursion. I also implemented a simplistic graphical user interface, and used a model-view controller (MVC) architecture.
Microsoft Paint Clone

A project for CSC207 - Software Design at UofT.

For this assignment, I worked in a group of four and used the SCRUM methodology to collaborate and advance the project. We implemented various features, such as shapes, text, polylines, squiggles, undo/redo, save/load, and more.

This project taught me a lot about what it means to work in a team, and how to divide work efficiently. I also learned GUI programming in Java, with Graphics2D, JavaFX, Swing, and AWT.

Since this is an assignment, the source code is not available online.
Randomized Text Generator

As indicated by the title, this simple project attempts to produce text by estimating the probability of a given word occurring after another in a text file fed to it as input.

While the text produced often makes no sense, I found the project fun to work with, and learned a great deal about Markov chains, which I used to implement the program.
